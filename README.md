# 🌌 NASA Universe Explorer

A minimalist web application built to explore the cosmos. This project interfaces with **NASA's Astronomy Picture of the Day (APOD) API** to fetch celestial data based on user-selected dates.

## ✨ Key Features
* **Dynamic API Integration:** Real-time data fetching using the NASA Planetary API.
* **Aesthetic UI:** Minimalist glass-morphism design with a focus on high-quality space imagery.
* **Responsive Layout:** Side-by-side view for desktop, optimized for readability and visual impact.
* **Smart Date Locking:** Calendar restrictions ensure users can only explore dates up to the current day.
* **Video Support:** Automatically detects and handles video content with a fallback link.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Flexbox & Glass-morphism)
* **Scripting:** JavaScript (Async/Await & Fetch API)
* **Source:** NASA Open APIs
